## Conversation
### Hanzi

今天中午，马克刚洗完碗，就发现水池堵了。他只好给房东家打了一个电话。

| 马克：      | 你好，请问王先生在吗？ |
| 房东的母亲：| 他刚出去。您是哪位？ |
| 马克：      | 我是马克。您知道他什么时候回来吗？ |
| 房东的母亲：| 大概二十分钟以后就回来，你过半个小时再打靶。 |
| 马克：      | 行。不好意思，打扰您了。 |
{: .conversation}

| 马克：      | 你好，我找王先生。 |
| 房东王先生：| 我就是，你是马克吧？|
| 马克：      | 对。王先生，不好意思，我想请您帮个忙。 |
| 王先生：    | 什么事？你说吧。 |
| 马克：      | 我的厨房有点问题，水池堵了。麻烦您过来看一下，可以吗？ |
| 王先生：    | 当然可以。我下午就过来，你在家吧？ |
| 马克：      | 可能会出去买东西。不过没关系，您可以进来。 |
| 王先生：    | 好的。 |
| 马克：      | 那就拜托您了。 |
| 王先生：    | 别客气。以后如果再需要我帮忙，可以随时给我打电话。 |
| 马克：      | 好的，谢谢您。 |
{: .conversation}
