# Color 颜色 (yánsè)
## Different colors

| 黑色 | hēisè   | black  |
| 白色 | báisè   | white  |
| 红色 | hóngsè  | red    |
| 绿色 | lǜsè    | green  |
| 蓝色 | lánsè   | blue   |
| 黄色 | huángsè | yellow |
| 棕色 | zōngsè  | brown  |

## Ask color of

To ask for the color of something, the question word 什么颜色(shénme yánsè) can be used in the following structure:  
__Subject + 是 + 什么颜色 + 的?__
